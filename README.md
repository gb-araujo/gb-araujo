## Sobre mim

Sou desenvolvedor de software júnior com atuação também em QA. Venho do suporte a sistemas, então gosto de entender o problema por inteiro: usuário, dado, log, bug, fluxo e impacto em produção.

Hoje trabalho principalmente com **C#**, **PHP**, **Kotlin Android** e **SQL**, além de estudar backend, mobile, automações e boas práticas para criar software mais confiável.

- Cursando Engenharia da Computação
